# Power-BI
An interactive Business Intelligence dashboard built using Power BI to analyze pizza sales data and generate actionable business insights. This project demonstrates data modeling, DAX calculations, and visual storytelling skills for data-driven decision-making.

Project Overview:
The dashboard analyzes key sales metrics such as revenue, order trends, product performance, and customer buying patterns. It helps identify top-selling pizzas, peak order times, and revenue contribution by category and size.

Key Insights Delivered:
Total Revenue, Total Orders & Average Order Value (AOV)
Daily & Monthly Sales Trends
Best & Worst Selling Pizzas
Sales Distribution by Pizza Size & Category
Peak Order Time Analysis

Technical Skills Demonstrated:
Data Cleaning & Transformation using Power Query
Data Modeling & Relationship Building
DAX Measures & Calculated Columns
KPI Cards & Interactive Visualizations
Drill-through Reports & Slicers

Tools & Technologies:
Power BI Desktop
DAX (Data Analysis Expressions)
Power Query
Excel / CSV Dataset
